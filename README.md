# ProjetoTsrnaGora
 Projeto de site para campanha de RPG de mesa Tsrna Gora. Criada e narrada por mim usando o [Sistema Gumshoe](https://en.wikipedia.org/wiki/Gumshoe_System) do designer [Robin Laws](https://en.wikipedia.org/wiki/Robin_Laws).

### Ideia:
A ideia é que o site se pareça com um site de agência de viagens mostrando este país fictício do Leste Europeu, suas belas paisagens, cultura e história. Como se trata de uma obra de ficção e um cenário de Role Playing Game de mesa, os textos tem uma mistura de texto publicitário com indicios de tramas e subtramas que serão usadas /encontradas na narrativa para que os jogadores, interpretando seus personagens, possam experenciar.
